Para que funcione la conexion con a bbdd tioenes que instalar el odbc que va adjunto en el zip.
En el caso de que el .pbix quede como zip tienes que cambiarle la extension, con la opción de "Mostrar extensiones de nombre de archivo" en windows le cambias la extension de .pbix.zip a .pbix (basta con borrar el .zip).
Despues de instalar el odbc no debería haber problemas para conectar con la bbdd desde powerbi.
Para actualizar los datos en el powerbi solo tienes que darle al boton de actualizar y te va a pedir las credenciales:
